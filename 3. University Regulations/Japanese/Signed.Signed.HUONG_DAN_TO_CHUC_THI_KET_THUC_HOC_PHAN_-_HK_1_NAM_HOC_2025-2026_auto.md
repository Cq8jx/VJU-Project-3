# 2025-2026年度第1学期 科目修了試験実施に関するガイダンス

2025年7月30日付の日越大学2025-2026年度学修課程スケジュール策定に関する公文書第840/ĐHVN-ĐT号に基づき、

2025年10月1日付の日越大学試験業務に関する規定の公布に関する決定第1132/QĐ-ĐHVN号に基づき、

教育の質を向上させ、学修課程スケジュールに則り科目修了試験を適正に実施するため、日越大学は各部署、教職員、学生に対し、2025-2026年度第1学期の成績評価及び科目修了試験の実施について、以下の通り指導します。

## 1. 実施原則

- 科目修了試験の実施において、誠実性、正確性、公平性、透明性を確保すること。
- 科目修了試験結果の評価において、学習者の権利を保障すること。
- 科目の学習到達目標の達成度を評価できる要件を確保すること。
- 日越大学及び外部機関（ハノイ国家大学体育・スポーツ教育センター、ハノイ国家大学デジタル教育・試験研究所、ハノイ国家大学外国語大学を含む）の2025-2026年度第1学期の学修課程の進捗及び計画が適切に実施されることを確保すること。

## 2. 科目修了試験の実施

### 2.1. 科目修了試験の実施期間

2025-2026年度第1学期の科目修了試験は、日越大学の学修計画に基づき、学士課程及び修士課程の各学年に対し、以下の期間で実施されます（表1参照）。

### 2.2. 試験実施プロセス

2025-2026年度第1学期の科目修了試験の実施業務には、試験計画の策定と日程の通知、試験問題作成と採点、試験監督官（CBCT）の配置、成績入力と試験結果の通知、及び試験の再採点が含まれます。具体的なプロセスは別添2に詳述します。

### 2.3. 試験問題及び解答の作成要件

- 科目修了試験問題は、当該科目の学習到達目標（CĐR）を評価できるものであること。
- 試験問題は、試験問題の回収が義務付けられている科目であっても、過去の学期試験、小テスト、演習問題、事前に学習者に配布された復習問題と同一の内容を繰り返してはならない。
- 同一科目が複数のクラスで開講される場合、共通の試験問題を作成し、事前に全担当教員及びプログラムディレクター／学部長が試験問題の要件と基本的内容について合意する必要がある（特別な場合は学長室の承認を得なければならない）。
- 試験問題バンクが既に存在する科目については、試験・品質保証室長が承認済みの試験問題バンクに基づいて試験問題を編集する。
- レポート、課題、報告書の形式で試験を行う場合、提出期限及び提出方法は科目担当教員が提案し、プログラムディレクター／学部長を通じて決定するが、大学の学習及び試験計画に準拠しなければならない。科目担当教員は、計画の実施、レポート等の回収・採点、試験問題・評価基準・レポート・報告書の提出、及び科目修了試験成績表の試験・品質保証室への提出時期について責任を負う。学習者は教員の指示に正しく従わなければならない。
- 口頭試験形式の場合：各試験問題には、設問ごとの明確な配点基準を設け、完全性、正確性、総合性、応用性、各試験問題間の難易度の均一性、単位数及び科目修了試験時間との適合性を確保しなければならない。
- 科目修了試験の試験時間：
    + 記述式または記述式と選択式の併用形式：1単位あたり30分とするが、4単位以上の科目については120分を超えないこと。
    + 選択式形式：1単位あたり20分とするが、4単位以上の科目については90分を超えないこと。
    + 口頭試験形式：準備時間は15分、回答発表及び教員からの追加質問の時間は約7～10分とする。準備時間が15分を超える科目は、科目シラバスに具体的に規定されなければならない。
- 試験問題には、配点基準及び明確な採点基準を伴う解答を添付しなければならない。採点は10点満点とし、小数点以下第1位まで採点する。選択式試験や外国語試験では、異なる配点基準で採点した後、10点満点に換算することができる（別添3、4の様式参照）。
- 試験問題と解答は別々の封筒に入れ、封をして割印を押す。各封筒には、試験問題の上部と同様の情報を明記し、作成者の氏名と電話番号を記載する。解答作成者は、解答に署名し氏名を明記しなければならない（添付様式に従う）。必要に応じて、大学は学習者が自己の答案を確認できるよう解答を公表する。
- 1試験時限あたりの試験問題数：
    + 記述式、記述式と選択式の併用形式：1試験時限あたり少なくとも2種類の問題を作成する。
    + 選択式形式：筆記試験の場合は各問題に少なくとも2つの問題コードを、コンピュータ試験の場合は少なくとも4つの問題コードを設けなければならない。
    + 口頭試験形式：試験問題は一式のセット（最低10問）として作成し、学習者が試験時間中にくじ引きで選び回答する形式とする。
- 科目修了試験問題の形式：
    + 試験問題は、A4用紙片面に、黒または濃い青のインクで、明確にタイプまたは手書きし、用紙の端から少なくとも2cmの余白を設けること。
    + 試験問題の上部には、科目名、科目コード、単位数、問題番号、試験時間を明記すること。
    + 試験問題の下部には、学習者が資料の使用を許可される場合は、その旨を明記すること。
    + 選択式試験の場合、選択式解答用紙の様式を選択式試験問題に添付する（別添3c参照）。

### 2.4. 採点に関する要件

- 採点官の配置：学部長／プログラムディレクターは、科目修了試験の答案を採点する教員を配置する責任を負い、各科目が少なくとも2名の教員によって採点されることを保証しなければならない。採点業務は、厳格、客観的、正確かつ公正に実施されなければならない。
- 採点及び成績提出のプロセス
    + 紙媒体での採点（記述式、記述式と選択式の併用、及び選択式試験の答案）。
        * 試験・品質保証室は、教員が採点を行うための部屋を準備し、採点の進捗状況を監督する責任を負う。
        * 科目の試験実施日の翌日から、採点を担当する教員は試験・品質保証室で答案を受け取ることができる。採点官は、科目の試験実施後、遅くとも2週間（10営業日）以内に、採点済みの答案と成績表を試験・品質保証室に提出しなければならない。
        * 科目修了試験の答案採点は、試験・品質保証室に提出された公式の解答と配点基準に従わなければならない。教員は答案に直接採点し、10点満点、小数点以下第1位まで採点する。答案には2名の採点教員の署名が必要である。誤りが発見された場合、教員は成績修正の申請を行い、修正理由を明記する。
    + コンピュータ室での採点
        * コンピュータ上で直接採点される科目の場合：試験時間終了後、学習者は作業を停止し、試験監督官が結果を確認するまでその場に留まる。試験監督官はコンピュータ画面に表示された試験結果を成績表に記録する。学習者は成績表の結果を再確認した上で、受験者名簿に署名する。試験監督官は、成績表に記載された試験結果の正確性について責任を負う。成績表は3部作成し、正本を試験・品質保証室に、1部を学部に、1部を教育・学生部に保管する。
        * 試験室で直接採点できない科目の場合：試験監督官は、各試験時限終了直後に、学習者の答案をCDまたはUSBに保存する責任を負う。試験終了後直ちに、試験監督官は受験者名簿、および学習者の答案を含むCDまたはUSBを試験・品質保証室に引き渡す。試験・品質保証室は、規定に従い、採点を担当する職員に学習者の答案を含むCDまたはUSBを引き渡す。
        * 科目修了試験の答案採点は、試験・品質保証室に提出された公式の解答と配点基準に従わなければならない。教員は採点票に採点し、科目の試験実施後、遅くとも2週間（10営業日）以内に、答案を含むCDまたはUSB、採点票及び成績表を試験・品質保証室に返却する。
    + プレゼンテーション、レポートの採点：科目修了試験の答案採点は、試験・品質保証室に提出された公式の解答と配点基準に従わなければならない。教員は採点票に採点し、採点票、答案、成績表を科目の試験実施後、遅くとも2週間（10営業日）以内に試験・品質保証室に提出する。
    + 口頭試験の採点：科目修了試験の答案採点は、試験・品質保証室に提出された公式の解答と配点基準に従わなければならない。学習者の回答が録音されない口頭試験形式の場合、試験結果は試験終了後直ちに学習者に公表され、科目修了試験日から遅くとも1営業日以内に試験・品質保証室に提出されなければならない。

### 2.5. 再採点試験結果の処理

- 学習者から再採点要求の申し出を受け取った後、試験・品質保証室は直ちに再採点対象の答案を取り出し、教育担当部署に通知する。学部長／プログラムディレクターは採点官を配置し、最大7営業日以内に再採点を行う。共通科目については、ハノイ国家大学の規定に従う。
- 再採点結果の処理は、初回採点と同様に行う。
- 再採点後の点数が初回採点（公表済み）の点数と0.5点から1.0点の範囲で異なる場合、直ちに点数を修正する。
- 再採点後の点数が初回採点の点数と1.0点以上異なる場合、学部長／プログラムディレクターは初回採点官とは別の職員を配置し、初回採点教員と2回目採点教員との間で直接対話を行い、再採点後の点数を統一して決定し、議事録を作成する。
- 成績処理の過程で、関係者が意図的な違反の兆候を発見した場合、監査・法務部門に報告し、処理を依頼する。
- 再採点結果は、試験・品質保証室が規定に従って処理し、再採点期間終了後、遅くとも3営業日以内に教育・学生部に送付され、学習者に通知される。

## 3. 各部署及び個人の責任

### 3.1. 試験・品質保証室

- 授業を調整する各部署から科目修了試験の形式に関する情報を集約する。
- 科目修了試験の日程を作成する。
- 受験者名簿を作成し、規定の期間内に試験室を配置する。
- 規定の期間内にすべての関係者に試験日程を通知する。
- 試験監督官（CBCT）の派遣リストを作成し、関係部署に送付し、規定の期間内に試験監督官の配置リストを提出するよう督促する。
- 採点基準表、採点済みの部分評価の答案例、期末成績表及び科目修了試験の答案を受領し保管する。保管方法と期間は、教育訓練省及びハノイ国家大学の現行の指導と規定に従う。
- 教員から試験問題、解答を受領し、試験問題バンク、科目修了試験問題を文書（印刷版）及びコンピュータデータファイル（データファイル）の形式で管理する。
- 大学の試験業務に関するプロセス、スケジュール及び規定に従い、科目修了試験問題のシャッフル、抽選、印刷、複製を実施し、法律の規定に従い機密保持体制を遵守する。
- 試験室の準備（試験室、待機室の予約、受験者名簿の人数に応じた机と椅子の配置を警備員に報告）。
- 採点室の準備、試験問題、解答用紙／マークシートの引き渡し、及び試験監督官との答案の受け渡しを行う。答案、科目修了試験成績表、及び採点業務に関する指導文書、規定を引き渡す。関係部署に対し、規定に従った採点、答案返却、成績表提出を督促する。
- 学習者に再採点申請の期間とプロセスを通知する。学習者からの申請があった場合の科目修了試験の再採点の窓口となる。

### 3.2. 教育・学生部

- 教育・学生部が調整する科目について、教員が期末試験の形式を確認するよう監督し、督促する。
- 各学期における各科目の部分評価成績表を学部、教育プログラム、教員が提出するよう監督し、督促する。
- 教員に成績入力ソフトウェアのアカウントを提供し、成績入力ソフトウェアのアカウントに関する問題を処理する。大学の現行の成績入力ソフトウェア上で部分評価成績の更新時期、部分評価成績表の提出時期を教員に通知する。
- 学生と教員の間で生じる部分評価成績データに関する問題を処理する。
- 試験・品質保証室から科目修了試験の成績データを受領し、教育管理ソフトウェア及び教育情報ポータルに科目評価成績データを更新し、学習者が把握できるようにする。

### 3.3. その他の事務部署

- 計画・財務部：試験・品質保証室と協力して試験の予算を準備し、試験・品質保証室に手数料、授業料未納の学習者リストを提供する。
- 事務局：監査・法務部門の職務権限に基づき、試験業務の実施プロセスを検査、監督する。関係者と協力し、業務プロセスが円滑に実施されるよう助言、指導、連携を行う。
- 教室、実験室、コンピュータ室を担当する部署及び個人は、自らが管理する施設を使用する試験の際の施設・設備について責任を負う。
- 各事務部署の責任者は、試験・品質保証室の要請に応じて試験監督官を派遣する責任を負う。

### 3.4. 学部長／プログラムディレクター、共同ディレクター

- 試験業務に関する規定及び科目修了試験の要件内容を、全教職員及び学習者に周知徹底する。
- 試験監督業務に参加する教員、研究員または学部アシスタントを試験監督官として派遣する。
- 学部が直接管理する科目について、科目担当教員、試験問題作成者、採点者を配置する。

### 3.5. 教員

- 受験資格のない学習者のリストを試験・品質保証室に送付する。
- 規定の期間内に試験問題、解答／採点基準、科目修了試験の成績を試験・品質保証室に提出する。
- 成績入力システムに科目修了試験の成績を入力する。
- 規定に従い再採点を行う（該当する場合）。

### 3.6. 試験監督官

- 責任感を持ち、試験監督業務に関する規定を十分に理解する。
- 別添5に記載の科目修了試験監督プロセスを正しく実行する。

### 3.7. 学生、院生

- 期限内に授業料を納付する。
- 試験で良い結果が得られるよう、自主的に学習し、試験勉強に励む。
- 別添7に規定されている試験室における受験者の責任を正しく履行する。
